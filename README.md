# catnet

catnet is the official command-line interface for the CatNet ecosystem.

It is designed for operators, sysadmins, labs, automation pipelines, and repeatable technical workflows. The CLI uses catnet-core as its execution engine.

## Goals
- Provide a clean and scriptable command-line experience.
- Expose stable operational workflows.
- Support structured output and automation.
- Remain lightweight and dependable.

## Planned commands
- catnet discover
- catnet scan
- catnet export
- catnet profile
- catnet version

## Status
Bootstrap phase. Command surface may evolve before the first stable release.
